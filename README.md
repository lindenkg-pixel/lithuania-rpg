# リトアニア音楽紀行

AKクワイア向け、リトアニアの音楽の祭典「ダイヌシュベンテ」を巡る感動RPG。
スマホブラウザで遊べます。

## 遊び方

- 十字キーで移動
- A: 決定 / 話す / 調べる
- B: 戻る / メッセージスキップ
- 緑の濃い草むらを歩くと戦闘が発生
- 「歌う」コマンドで感動ダメージ。敵が泣いたら勝利

## 開発環境セットアップ

### 必要なもの
- ブラウザ（Chrome/Safari推奨）
- ローカルサーバー（`file://` 直開きだと制約あり）

### 起動

```bash
cd C:\Users\kei_x\works\lithuania-rpg
python -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開く。

### スマホ実機で確認

PCと同じWi-Fiにスマホを繋ぎ、PCのローカルIP（`ipconfig` で確認）で:
```
http://192.168.x.x:8000/
```
