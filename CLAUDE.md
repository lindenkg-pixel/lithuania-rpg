# lithuania-rpg — リトアニア音楽紀行

AKクワイア向けの、リトアニア各都市を音楽と一緒に旅するドラクエ風RPG。
スマホブラウザで遊べる単一HTMLゲーム。

## プロジェクトの目的

AKクワイアの方々に「楽しんでもらいながらリトアニアと言語を学んでもらう」ことが最優先。
教材ではなく**ゲームとして遊んで完走できる**こと。クリアまで2-3時間以上想定（厳密ではない）。

## ストーリー

主人公はAKクワイアの一員。リトアニアの音楽祭典「ダイヌシュベンテ」で歌うために来たが、
歌詞をすっかり忘れてしまった。リトアニア国内に散らばった4曲分の歌詞ピース（計20個）を集めて、
祭典で全曲歌い上げる感動ストーリー。

通底テーマは伝説の音楽家・画家 M.K.チョルリョーニス。要所で精霊として現れヒントをくれる。

サブタイトル: 「〜失われた歌詞のピース〜」

## 配布方針

完成したらGitHub Pagesでホスト。AKクワイアの方々は配布URLにスマホでアクセスして遊ぶ。
アカウント不要。インストール不要。スマホブラウザで完結。**商用ではない（課金なし）**。

## 技術スタック

- **言語**: バニラJS（フレームワーク不使用）
- **描画**: HTML5 Canvas（ドラクエ風ドット絵、`imageSmoothingEnabled = false`）
- **タイル**: 32px × 32px、マップは32×32タイル、画面はカメラ追従式
- **入力**: タッチ（十字キー＋A/Bボタン）＋ キーボード（矢印/WASD/Z/X）
- **保存**: localStorage（オートセーブ。**リリース後はスキーマ変更しない方針**）
- **音**: Web Audio API合成音 + フリー素材（CC0/CC-BY中心。商用ではないので大半利用可）
- **依存**: なし（外部ライブラリゼロ）

## ゲームデザインのコア仕様

### マップ
- タイル: **32px × 32px**
- マップサイズ: **32×32タイル**（現在の4倍面積）
- 描画: 画面はカメラ追従、プレイヤー中心の視野
- 各都市は**実物の地理関係に寄せた配置**
- 都市間移動: メイン都市のピース取得後、駅で次の都市行き切符が解禁 → 電車演出（無料）
- 既訪都市への再訪: みほさんに「○○に戻りたい」と頼むとワープ

### タイル定義（暫定。都市追加時に拡張）
```
0: 草地        1: 道           2: 建物壁        3: 木
4: 水          5: 建物入口      6: ゲディミナス塔 7: 夜明けの門
8: 聖アンナ教会 9: 草むら(エンカウント)
```
通行不可: 2, 3, 4, 6, 7, 8 / ランダムエンカウント: 9（確率18%）

### パーティ
- **3人パーティ固定**（主人公＋親しい友達2人）
- ヴィリニュス序盤、あきちゃんが「2人連れて行きなよ」と紹介して仲間2人加入
- 名前は3人とも自由入力（プレイヤーが大切な人を入れる）
- 見た目: **男性4種＋女性8種＝計12種類**のプリセットから選択
- 全員職業「歌手」、**共通レベル**

### 戦闘
- 「歌う」コマンド = 攻撃。曲ごとにMP消費・威力が違う
- 敵HP = 感動メーター。0で「敵が泣いて」勝利
- 行動順: 主人公 → 仲間1 → 仲間2 → 敵
- **主人公のみプレイヤー操作、仲間2人はAI自動行動**
- 敵の攻撃対象: ランダム
- ボスは「感動耐性」でダメージ -3 補正

### 戦闘不能と復活
- HP0 = 「泣き崩れて使い物にならない」状態
- **戦闘中**: 他の生存メンバーが「励ます」コマンド（1ターン消費・復活のみ）で現実に引き戻す
- **全員泣き崩れ**: 全滅 = 直近の宿屋（みほさん）に戻る

### 合体技「エホーマイ」
- AKクワイアでレッスン開始前に唱える呪文がモチーフ
- 3人全員生存時のみ発動可能
- MP消費: 主人公6 / 仲間1×2 = 合計8
- 効果: 次のターンから3ターン、歌の威力2倍
- 演出: 「3人で手を繋ぎ、エホーマイ！と唱えた！」+ 画面が光る

### キャラ成長
- **共通レベル制**（パーティ全体で1つのレベル）

- nextXp = floor(現在のnextXp * 1.6) で逓増
- **クリア時 Lv15 想定**

### リトアニア語学習
- 序盤: あいさつ3つ（Labas / Ačiū / Kaip sekasi?）
- 中盤: 自己紹介・買い物・道案内（Aš esu... / Kiek kainuoja? / Kur yra...?）
- 終盤: 感情表現・歌詞理解・文化談義
- 都市進行に合わせて3択の難度上昇
- 正解 → 歌詞ピース or ヒント / 不正解 → 「うまく通じなかった」
- 試験ではなく、**自然に何度も出会うことで定着**させる方針

### セーブ
- **オートセーブ**（場所移動・戦闘終了時・ピース取得時）
- 起動時「はじめから／つづきから」選択
- **リリース後はデータ構造を変えない方針**（古いセーブとの互換性問題を避ける）
- セーブには `version: 1` のスキーマ番号を持たせておき、将来の拡張余地は確保

### みほさん（宿屋システム）
- 各都市の同じ位置（宿屋）に配置
- 機能: HP/MP全回復・セーブ・復活・他都市ワープ（既訪のみ）
- AKクワイア主催あきちゃんの奥様、お母さん的存在

### ヴィリニュス再訪の作り込み（Phase 6 強化方針 / 将来タスク）

ストーリー的にヴィリニュスは他都市より厚く濃く描く方針。前半攻略後、後半で再訪したときに体験差分を多く出す:

- **NPC構成の変化**: 前半に居た人は転居/旅立ち、新しい顔ぶれが街に増える（ルツィア成長後の再登場、ユウコヤマサキ着任、ギンターレ来訪）
- **既存NPCの台詞差分**: 前半で会った人（マンタス・イエヴァ・あきちゃん・みほさん・大聖堂の司祭・古老）も再訪時はストーリー進行を反映した別台詞に
- **文化施設のイベント化**: 前半は BUILDING_INFO の解説パネルだけだった建物（聖アンナ教会・大聖堂・夜明けの門・ウジュピス橋・大統領官邸）で、再訪時のみ起動するイベント／歌詞ピース獲得を用意
- **マップオブジェクトの追加**: 再訪フラグ（state.flags.vilnius_revisit）でゲディミナス塔の上層階解放、Vingisパーク経路の出現
- **判定**: state.cityKey === 'vilnius' && state.flags.vilnius_revisit のとき別NPCテーブル/別 talkNPC 分岐に切り替える設計が綺麗
